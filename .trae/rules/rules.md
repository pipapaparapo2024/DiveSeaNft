# Frontend Test Project Rules

## Tech Stack
- Framework: Next.js (App Router)
- State Management: Redux Toolkit
- Styling: SCSS (Modules preferred)
- Animation: GSAP or Framer Motion
- UI: Adaptive/Responsive (Mobile, Tablet, Desktop)

## Coding Standards
- Use functional components and TypeScript.
- Follow a clean folder structure: /components, /store, /hooks, /styles, /services.
- For the Slider: Use the CoinGecko API (/nfts/list). Fetch data in a Thunk or Service.
- For Assets: Use random placeholder images for NFTs as per requirements.

## Style Guidelines
- strictly follow the Figma layout proportions and spacing.
- Implement smooth transitions using GSAP/Framer Motion.
